# Resultados de pruebas

Este directorio contiene o almacenará los artefactos generados por las ejecuciones de rendimiento.

## Archivos esperados

- `app.log`
- `smoke.json`
- `baseline.json`
- `load.json`
- `stress.json`
- `spike.json`
- `soak.json`

| Escenario | Propósito | Artefacto |

| Stress | punto de quiebre | `stress.json` |
